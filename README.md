# Whiteboard to Gantt — OpenClaw for Project Managers

From sticky notes to structured timelines. An interactive presentation showing how [OpenClaw](https://github.com/openclaw/openclaw) turns whiteboard chaos into actionable Gantt charts.

![Preview](assets/og-image.png)

## What is this?

An interactive slide deck (11 slides) that demonstrates the **whiteboard-to-gantt** workflow for project managers:

- 📸 Snap a photo of your whiteboard
- 🧠 OpenClaw's vision model reads tasks, dependencies, owners, dates
- 📋 Structured data extracted automatically
- 📊 Gantt chart pushed to OpenProject, Jira, or exported as HTML/CSV

## Features

- **Single HTML file** — no build step, no dependencies
- **Dark/light theme** toggle
- **English/French** language toggle
- **Keyboard, click, and touch** navigation
- **Slide menu** for quick jumping
- **Terminal aesthetic** with JetBrains Mono

## Usage

Open `index.html` in a browser. Navigate with:

- **Arrow keys** or **Space** — next/previous slide
- **Click** left/right halves of the screen
- **Swipe** on touch devices
- **F** for fullscreen
- **☰ Menu** button for slide overview

## Tech Stack

- Vanilla HTML + CSS + JS
- JetBrains Mono (Google Fonts)
- CSS custom properties for theming
- Zero dependencies
